# Meeting notes

Call notes, debriefs, and follow-up tracking. One file per meeting.

## Naming

`YYYY-MM-DD-municipality-or-counterparty.md`

Examples:
- `2026-05-05-mississauga-jeff-richard.md`
- `2026-05-12-ajax-clerk-followup.md`

## Template

```markdown
# [Counterparty] — [Date]

**Attendees**
- Us: [names]
- Them: [names + roles]

**Format**
[20-min intro / pitch / working session / follow-up]

## What happened

[Narrative summary, in the order things came up. Quote them where the
phrasing matters.]

## Reactions and tells

[How they responded. Objections raised. Body language if relevant.
Distinguish "they liked it" from "they understood it" — those are different.]

## Pricing / commercial signals

[Anything they said about budget, procurement path, decision authority,
timeline.]

## Open questions / next step

[Concrete next action, owner, expected timing.]

## Quotes worth saving

[One-liners that capture how they think — useful for landing copy
and future pitches.]
```
